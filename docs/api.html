API Reference

AsyncPocketOptionClient

  • AsyncPocketOptionClient(SSID, is_demo=True, enable_logging=False)
    Create a new client instance.
  • await client.connect()
    Connect to Pocket Option using your SSID.
  • await client.disconnect()
    Disconnect from the server.
  • await client.get_balance()
    Get your account balance and currency.
  • await client.get_candles(asset, timeframe)
    Get candle data for a symbol and timeframe.
  • await client.get_candles_dataframe(asset, timeframe)
    Get candle data as a pandas DataFrame.
  • await client.place_order(asset, amount, direction, duration)
    Place a CALL or PUT order.
  • await client.get_active_orders()
    List your currently active orders.
  • await client.check_order_result(order_id)
    Check the result of an order (win/loss/pending).
  • await client.check_win(order_id)
    Check if an order was a win or loss.
  • await client.get_connection_stats()
    Get connection statistics and status.

OrderDirection

  • OrderDirection.CALL - Place a CALL order
  • OrderDirection.PUT - Place a PUT order
